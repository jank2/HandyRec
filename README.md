# HandyRec
Multi-track recorder for musicians

# User Manual for HandyRec

## About the application
**HandyRec** is a cross-platform recording application designed specifically for musicians who need to capture their musical ideas quickly and easily.

The main philosophy of the program is **speed and efficiency**. Thanks to a sophisticated system of keyboard shortcuts, you can control the entire application without having to take your hands off the keyboard or instrument. This allows you to focus on what is most important - your music.

The application offers 8 independent recording tracks, an integrated metronome with advanced settings, and project management options (saving, loading, exporting).

---

## Installing the program

### Linux (Flatpak)
To install on Linux, use the Flatpak package. Download the `.flatpak` file and install it with the `flatpak install <file>.flatpak` command or by double-clicking the file in the software manager.

### Windows (Installer)
For Windows, download the `.exe` installation file. Run it and follow the installation wizard.

### macOS (App and DMG)
On macOS, you can use the `.dmg` disk image. Open the downloaded `.dmg` file and drag the application to the Applications folder. Alternatively, you can directly run the `.app` package.

## Overview of functions and keyboard shortcuts

Below you will find a description of the individual functions of the application along with their keyboard shortcuts to speed up your work as much as possible.

### ‚èØÔ∏è Basic controls (Transport)
Basic functions for playback and recording.

| Function | Keyboard shortcut | Description |
| :--- | :--- | :--- |
| **Play / Pause** | `Spacebar` | Starts or pauses project playback. |
| **Record** | `Ctrl + Spacebar` | Starts recording on tracks that are "armed" (Arm). |
| **Stop** | *Mouse (Stop button)* | Stops playback and returns the cursor. |

### üéöÔ∏è Track Control (1-8)
Each of the 8 tracks has a number (1 to 8). Shortcuts always refer to a specific track number.
*You can also use the number keys above the letters on the numeric keypad.*

| Function | Shortcut | Description |
| :--- | :--- | :--- |
| **Arm** | `Number (1-8)` | Prepares the track for recording (red light). |
| **Mute** | `Ctrl + Number (1-8)` | Temporarily mutes the given track (M). |
| **Solo** | `Alt + Number (1-8)` | Plays only this track, mutes the others (S). |
| **Delete content** | `Ctrl + Alt + Number` | Deletes recorded audio in the given track (requires confirmation). |

### ‚è±Ô∏è Metronome & Timing
Tools for keeping the beat.

| Function | Shortcut | Description |
| :--- | :--- | :--- |
| **Metronome Mode** | `M` | Toggles modes: Off / Sound / Visual. |
| **Count-In** | `C` | Toggles the length of the countdown before recording (Off / 1 / 2 bars, etc.). |
| **Time Sig** | `B` | Changes the metronome time signature (e.g. 4/4, 3/4, 6/8). |

### üõ†Ô∏è Tools
Useful tools for musicians.

| Function | Shortcut | Description |
| :--- | :--- | :--- |
| **Tuner** | `T` | Opens the chromatic tuner. Supports playing notes (Space) and changing strings (Tab). |
| **Auto-BPM** | `A` | Opens the tempo detector. Analyzes the recording on track 1 and sets the metronome accordingly. |
